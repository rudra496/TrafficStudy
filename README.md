# TrafficStudy

TrafficStudy is an **educational and research-focused project**
designed to help learners understand **network traffic behavior**
and **defensive response mechanisms** such as rate limiting and blocking.

This repository may include historical traffic-generation code
for **controlled, permission-based environments**, along with
offline educational tools for analysis and learning.
